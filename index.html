Usage ğŸ–±ï¸Â¶

!showcase video

mov-cli comes packaged with a CLI interface via the mov-cli command you can use in your respective terminal.

[!NOTE] You may notice mov-cli doesnâ€™t ship with any scrapers (or previously known as providers) by default, this is because v4 is plugin-based and scrapers are now part of plugins that must be chosen to be installed. Find out how to do so at the wiki.

  1. Install the scraper plugin of your choice. Visit this wiki page on how to do so and the mov-cli-plugin topic for a list of third-party mov-cli plugins.

mov-cli -e

Alternatively, you may also edit by manually opening the config file. See this Wiki page on that.

[mov-cli.plugins]
youtube = "mov-cli-youtube"

  1. Scrape away!

mov-cli -s youtube blender studio

The command above searches for blender studio with our youtube plugin, however once again mov-cli is plugin based and there are many of them in the wild. ğŸ˜‰
